# 🤝 Code of Conduct

*Building tools that protect — not harm.*

</div>

---

## Our Pledge

We are committed to building tools that advance **defensive security through offensive testing**. This project exists to help security professionals protect systems, not to enable harm.

Everyone who contributes to or uses this project is expected to uphold these standards — in issues, pull requests, discussions, and beyond.

---

## ✅ Expected Behavior

- Use these tools **only with proper authorization**
- Report vulnerabilities **responsibly** and to the right parties
- Help others learn offensive security in **ethical contexts**
- Provide **constructive feedback** in issues and PRs
- Respect the **privacy and security** of others at all times

---

## ❌ Unacceptable Behavior

- Using or promoting these tools for **unauthorized access**
- Sharing results from **unauthorized testing**
- **Harassment, discrimination, or personal attacks** in project spaces
- Publishing others' **private information** without consent
- Any conduct that **violates applicable laws**

---

## ⚙️ Enforcement

Project maintainers may:

- Remove, edit, or reject contributions that violate this code of conduct
- Issue warnings for first-time or minor violations
- Ban contributors for repeated or severe violations

All enforcement decisions are at the discretion of the project maintainers.

---

## 🚨 Reporting

> Report violations to **security@maeitsec.github.io**

All reports will be reviewed promptly and handled **confidentially**. We take every report seriously.
